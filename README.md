# Sports Motion Tracking System

## Overview
This project develops a modular sports motion tracking system using an ESP32 microcontroller and an IMU sensor. The system captures motion data and applies machine learning to classify sports movements.

## Features
- IMU-based motion sensing (acceleration + rotation)
- Modular 3D-printed mounts (wrist, racket, club)
- Wireless / serial data collection
- Machine learning classification

## Hardware
- ESP32
- MPU6050 IMU
- 3D printed mounts

## Software
- Arduino (ESP32 firmware)
- Python (data processing + ML)
- TinkerCAD / CAD (mount design)

## Folder Structure
- firmware/ → ESP32 code
- ml/ → machine learning models
- cad/ → 3D designs
- data/ → collected datasets
